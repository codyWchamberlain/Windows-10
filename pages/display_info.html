Display

Computer Only Mode

This mode will do just as it says, display only on the workstation. Whether you have an external monitor plugged in or not, the display will remain on the workstation

Duplicate Mode

This Mode will copy your desktop to each screen. Everything you have on one screen, will appear on the other

Extend Mode

This mode will extend your desktop to each monitor. You can drag windows to each monitor to make multitasking easy.

Projector Only Mode

This mode will display on the extneral Monitor only. Your workstation monitor will remain blank whether it is turned on or not
